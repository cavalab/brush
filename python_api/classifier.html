BrushClassifier#

class brush.estimator.BrushClassifier(**kwargs)[source]#

Brush for classification.

For options, see BrushEstimator.

Examples

>>> import pandas as pd
>>> df = pd.read_csv('docs/examples/datasets/d_analcatdata_aids.csv')
>>> X = df.drop(columns='target')
>>> y = df['target']
>>> from brush import BrushClassifier
>>> est = BrushClassifier()
>>> est.fit(X,y)
>>> print('score:', est.score(X,y))

predict_proba(X)[source]#

Predict class probabilities for X.

Parameters:
X{array-like, sparse matrix} of shape (n_samples, n_features)

The input samples. Internally, it will be converted to dtype=np.float32.

Returns:
pndarray of shape (n_samples, n_classes)

The class probabilities of the input samples. The order of the classes corresponds to that in the attribute classes_.
